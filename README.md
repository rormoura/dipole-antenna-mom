# Dipole Antennas - Method of Moments (MoM)

This repository contains a computational numerical analysis of the properties of dipole antennas using the Method of Moments (MoM). The project was developed as part of the Antennas course at the Centro de Informatica, Universidade Federal de Pernambuco (UFPE).

## Objective

The main objective is to perform a numerical analysis of linear dipole antennas using the Method of Moments. It includes the calculation of elements for the Impedance (Z) and Voltage (V) matrices, applying triangular basis functions.

## Geometry and Setup

The problem models a linear dipole antenna with the following characteristics:
* Length: L, aligned along the z-axis.
* Feed: Center-fed with voltage V and current I.
* Assumptions: The wire radius a is much smaller than the wavelength, and the antenna material is a perfect conductor.
